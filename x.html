Technical Details

Your ProjLib Beta is a client-side web proxy that works by loading websites through an iframe element. Since it doesn't use a server-side component, there are no authentication requirements or usage restrictions.

The proxy works best with websites that don't implement iframe blocking techniques like X-Frame-Options or Content Security Policy (CSP) headers.
